## Par uzņēmumu un darba prasībām:
### Mikrotīkls





Es izvēlējos uzņēmumu SIA "Mikrotīkls",[^1] , jo tas ir moderns IT uzņēmums ar daudz 
karjeras izaugsmes iespējām un labu atalgojumu. Mikrotīkls arī nodrošina veselības apdrošināšanu, apmaksātas pusdienas un brīvu autostāvietu. Mikrotīkla biroji ir moderni, plaši un nodrošina visu nepieciešamo priekš darba.​

Biroji atrodas Ūnijas ielā 2, Rīgā.

### Mikrotīkla tehnoloģijas,[^1]

Mikrotīkls ražo un veido dažādas tehnoloģijas kā:
- *Mikrotik RouterOS* programmatūru
- *RouterBOARD* maršutētājus

Mikrotīkls izmanto C un C++ programmēšanas valodas un Linux, lai izstrādātu Mikrotik RouterOS.

### Prasības [^2]

Šī darba vakance pieprasa ļoti labas zināšanas C un C++  programmēšanas valodā [3], laba izpratne par OS kodola uzbūvi (OS kernel space) [4], pieredze kodola draiveru izstrādnē un teicamas latviešu valodas un angļu valodas zināšanas.

Šis darbs nodrošina mēneša bruto algu no **4000 – 10 000€ mēnesī**, atkarībā no pieredzes.

Šis ir backend programmēšanas darbs.

Pilna slodze.
